CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web agar terlihat lebih menarik dan rapi. Dengan CSS, kita dapat mengubah warna teks, ukuran huruf, jarak antar elemen, serta tata letak halaman sesuai kebutuhan. CSS biasanya digunakan bersamaan dengan HTML untuk mempercantik tampilan form, tombol, dan elemen lainnya sehingga halaman web menjadi lebih nyaman dilihat dan digunakan.
